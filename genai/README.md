# GenAI — WC2026 Face Match + Office Quiz

FastAPI service running on **port 8002**. Two features:
- **Face match** — upload a photo, get the World Cup 2026 player who looks most like you
- **Office quiz** — answer 4 questions, get a "dream office team" of WC2026 players

Swagger UI available at `http://localhost:8002/docs` once running.

---

## Prerequisites

- Python 3.13
- [uv](https://docs.astral.sh/uv/) installed
- OpenAI API key (only needed for the quiz feature)

---

## Setup

### 1. Install dependencies

From the **repo root**:

```bash
uv sync
```

### 2. Configure environment

```bash
cp genai/.env.example genai/.env
```

Edit `genai/.env` and fill in:

```env
OPENAI_API_KEY=sk-...      # required for quiz
STATS_PDF=                 # optional, only needed to re-run ETL
API_PORT=8002              # default, change if needed
```

### 3. Start the API

```bash
uv run genai-api
```

The API is ready when you see:
```
INFO:     Application startup complete.
```

→ `http://localhost:8002`
→ `http://localhost:8002/docs` (interactive Swagger UI)

---

## Frontend env var

Add this to `apps/web/.env`:

```env
VITE_GENAI_URL=http://localhost:8002
```

Then call endpoints using `import.meta.env.VITE_GENAI_URL` as the base URL.

## Quick test

```bash
# Players
curl http://localhost:8002/players/1

# Face image
curl http://localhost:8002/faces/1 -o face.jpg

# Face match
curl -X POST http://localhost:8002/match \
  -F "photo=@your_photo.jpg" \
  -F "model=clip" \
  -F "top=3"

# Quiz
curl -X POST http://localhost:8002/quiz/start \
  -H "Content-Type: application/json" \
  -d '{"role": "Soy desarrollador frontend, trabajo con React y me encargo de la UI"}'
curl -X POST http://localhost:8002/quiz/answer \
  -H "Content-Type: application/json" \
  -d '{"session_id": "<id from previous step>", "answer": "B"}'
```

## Data files (gitignored)

These are generated by the ETL pipeline and live in `genai/`:

| File | Description |
|------|-------------|
| `players.db` | SQLite with all 1165 players + stats |
| `embeddings*.npy` | Pre-computed face embedding matrices |
| `faces/` | Cropped player face images (300×300) |

They are already generated — do **not** delete them unless you want to re-run the full ETL pipeline.

To re-run ETL:
```bash
uv run genai-pipeline --stats-pdf genai/statsFifa.pdf
```
